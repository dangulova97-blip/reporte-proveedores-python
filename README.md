# Automatización de reporte con Python

Este proyecto procesa un archivo de datos de proveedores y genera resúmenes automáticos por área y estado

## Utiliza
- Python
- pandas

## Qué hace
- Lee un archivo Excel/CSV
- Limpia los datos
- Agrupa por área y estado
- Genera un resumen automático

## Cómo ejecutarlo
python reporting.py


## Ejemplo de resultado

El script genera un archivo Excel con las siguientes hojas:

- Resumen por área (ordenado de mayor a menor)
- Resumen por estado
- Resumen de pendientes por área
